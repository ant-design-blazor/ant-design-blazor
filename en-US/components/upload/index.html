Upload

Upload file by selecting or dragging.

When To Use

Uploading is the process of publishing information (web pages, text, pictures, video, etc.) to a remote server via a web page or upload tool.

  • When you need to upload one or more files.
  • When you need to show the process of uploading.
  • When you need to upload files by dragging and dropping.

Examples

Upload by clicking

Classic mode. File selection dialog pops up when upload button is clicked.

Avatar

Click to upload user's avatar, and validate size and format of picture with beforeUpload.

Pictures Wall

After users upload picture, the thumbnail will be shown in list. The upload button will disappear when count meets limitation.

Drag and Drop

You can drag files to a specific area, to upload. Alternatively, you can also upload by selecting.

We can upload serveral files at once in modern browsers by giving the input the multiple attribute.

Pictures with list style

If uploaded file is a picture, the thumbnail can be shown. IE8/9 do not support local thumbnail show. Please use thumbUrl instead.

Upload by Pasting

You can copy files to clipboard and paste them in the input box to upload.

Use native InputFile

You can use the native InputFile component to upload as file stream. The Upload component only display the FileList.

Default Files

Use defaultFileList for uploaded files when page init.

Complete control over file list

You can gain full control over filelist by configuring fileList. You can accomplish all kinds of customed functions. The following shows two circumstances:

  1. limit the number of uploaded files.

  2. read from response and show file link.

Upload directory

You can select and upload a whole directory. he thumbnail can be shown. IE8/9 do not support local thumbnail show. Please use thumbUrl instead.

Upload by Manually

After selecting the file, do not upload it. Press the Start button and then upload it

FAQs#

How to implement upload server side?

  • You can consult Asp.Net Core WebAPI or jQuery-File-Upload about how to implement server side upload interface.
  • There is a mock example of express in rc-upload.

Can I use FileStream?

FileStream is not implemented in this component, if you pursue uploading on the Server Side and omit WebAPI, you can try to use the official InputFile component (and beautify it with Antd's style)

I want to display download links.

Please set property url of each item in fileList to control content of link.

Antiforgery verification is enabled on the server. How to add AntiForgeryToken to the uploaded request?

The Upload component automatically looks for the ' ', if it is used in a static form, you do not need to do anything. If you are using the Blazor Server, you need to add the '' component to the static App.razor, Or through the official documentation The way mentioned in the manual creation of such hidden domains.

If the AntiForgeryToken is obtained through Ajax, add it to the upload request using the Data or Headers parameter.
