Tree树形控件

多层次的结构列表。

何时使用

文件夹、组织架构、生物分类、国家地区等等，世间万物的大多数结构都是树形结构。使用 树控件 可以完整展现其中的层级关系，并具有展开收起选择等交互功能。

Examples

基本

最简单的用法，展示可勾选，可选中，禁用，默认展开等功能。

拖动示例

将节点拖拽到其他节点内部或前后。

异步数据加载

点击展开节点，动态加载数据。

连接线

节点之间带连接线的树，常用于文件目录结构展示。使用 showLine 开启，可以用 switcherIcon 修改默认图标。

目录

内置的目录树，multiple 模式支持 ctrl(Windows) / command(Mac) 复选。

虚拟滚动

使用 height 属性则切换为虚拟滚动。

Node Selectable

使用SelectableExpression来动态设置节点是否可选择。

注意：也可以使用Selectable="false"来标记节点不可选择。

受控操作示例

受控操作示例

拖放节点精确控制

使用DraggableExpression和DroppableExpression精确控制拖放节点。

可搜索

可搜索的树。

可以使用 SearchExpression 来自定义匹配规则，也可以指定 HideUnmatched 来只显示匹配搜索值的节点（以及它们的直接父节点）。

自定义图标

可以针对不同的节点定制图标。

自定义展开/折叠图标

自定义展开/折叠图标。

Node Checkable

使用Checkable="false"来标记节点不可勾选。

注意：也可以使用CheckableExpression来动态设置节点是否可勾选。
