## zh-CN

使用`@bind-Value`设置或获取多选值。

> 注意：需要启用`EnableFlags`，并具有`FlagsAttribute`属性。

## en-US

Use `@bind-Value` to set or get multiple selected values.

> Note: Need to enable `EnableFlags` and have the `FlagsAttribute` applied to your enum.
