## zh-CN

使用 `OverlayService` 在任意位置显示上下文菜单。右键点击区域可以在鼠标位置展示菜单。

这个示例展示了如何使用 `OverlayService.OpenAsync()` 方法在指定坐标显示 Blazor 渲染片段。

## en-US

Use `OverlayService` to display context menus at any position. Right-click the area to show the menu at the mouse position.

This example demonstrates how to use the `OverlayService.OpenAsync()` method to display Blazor render fragments at specific coordinates.
