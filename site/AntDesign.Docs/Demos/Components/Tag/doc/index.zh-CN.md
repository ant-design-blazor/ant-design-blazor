进行标记和分类的小标签。

## 何时使用

- 用于标记事物的属性和维度。
- 进行分类。


## API

| 参数             | 说明                                         | 类型          | 默认值    |
| ---------------- | -------------------------------------------- | ------------- | --------- |
| Closable | 值标签是否可以关闭| boolean         |-       |
| Checkable | 值标签是否可以选择 | boolean         |-       |
| Checked | 标签是否关闭所对应的值 | boolean         |-       |
| CheckedChange | 点击标签时触发的回调 | function(e)         |-       |
| Color | 预设标签色 | string   | -         |
| PresetColor | 标签的预设颜色 | PresetColor   | -         |
| OnClose | 关闭时的回调     | function(e)         | -         |
| Visible | 是否显示标签 | boolean         |
| Icon | 设置图标  | string        | -         |
