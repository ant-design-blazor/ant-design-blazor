## zh-CN

使用 `IconTemplate` 参数可以自定义标签的图标部分，实现类似 shields.io 或 GitHub issue label 的双色标签效果。

## en-US

Use the `IconTemplate` parameter to customize the icon part of the tag, achieving a two-tone tag effect similar to shields.io or GitHub issue labels.
