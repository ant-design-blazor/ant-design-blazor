区隔内容的分割线。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。


## API

时间轴

| 参数             | 说明                                         | 类型          | 默认值    |
| ---------------- | -------------------------------------------- | ------------- | --------- |
| Dashed   | 是否虚线| boolean         |false|
| Orientation | 分割线标题的位置 `left`, `right`, `center` | string         |-       |
| Style | 分割线样式对象| CSS properties  | -  |
| Type | 水平还是垂直类型 `horizontal` or `vertical`| string | -  |
| Plain | 文字是否显示为普通正文样式| boolean  | -  |
