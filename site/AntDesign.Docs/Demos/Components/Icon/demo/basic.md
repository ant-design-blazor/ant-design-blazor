## zh-CN

通过`Type`属性设置图标名称，`Theme`属性设置主题，还可以通过设置 `Spin` 属性来实现动画旋转效果。

## en-US

Set icon name with `Type` property, set theme with `Theme` property. You can also specify the `Spin` property to show spinning animation.

<style>
.icons-list .anticon {
  margin-right: 6px;
  font-size: 24px;
}
</style>
