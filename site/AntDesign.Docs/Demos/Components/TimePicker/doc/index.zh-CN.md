输入或选择时间的控件。

## 何时使用

---

当用户需要输入一个时间，可以点击标准输入框，弹出时间面板进行选择。

## API

目前已经支持 4 种类型的值:

- DateTime
- DateTimeOffset
- DateOnly
- TimeOnly

时间选择器完成度较低，暂时只实现了以下功能：
autoFocus、bordered、className、defaultValue、
disabled、format、inputReadOnly、open、placeholder、
popupStyle、renderExtraFooter、value、onChange、onOpenChange
