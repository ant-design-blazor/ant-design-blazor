## zh-CN
设置 `AutoSize` 属性为 `true` 启用高度自动变化。当设置了 `MinRows` 或 `MaxRows` 时，会强制开启 `AutoSize`。


## en-US
Set `AutoSize` to `true` to enable autosizing. When `MinRows` or `MaxRows` is specified, `AutoSize` will be always `true`.
