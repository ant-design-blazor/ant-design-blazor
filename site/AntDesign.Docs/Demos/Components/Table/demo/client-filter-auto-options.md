## zh-CN

客户端数据源模式下，`Filterable` 列会自动从当前已加载数据生成筛选项（多选 checkbox）。

此示例使用前端分页：翻页不会重置筛选项和已选条件。

## en-US

In client-side data mode, `Filterable` columns automatically generate filter options (multi-select checkboxes) from loaded data.

This demo uses client-side pagination. Changing pages does not reset filter options or selected filters.
